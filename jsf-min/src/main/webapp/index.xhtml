JSF 2.2: Simple JSF

JSF establece el estandar para la construcción de UI basado en componentes web.

  • JSF trae componentes (tags) predefinidos y un api controlar sus eventos.
  • Puede ser utilizado como framework MVC: creación de formularios, control de eventos, validación de datos, invocación de una lógica de negocio y generación de resultados.
  • Tiene soporte para AJAX. También se puede utilizar jQuery o GWT(Google Web Toolkit).
